
# Get Comment by itemid

Get all of comments for an item .

**URL** : `/user/:id/item/:pid/comment/itemList`

**Method** : `GET`

**Auth required** : YES 

**Permissions required** : YES



## Success Response

**Condition** : If everything is OK.

**Code** : `200 Sucess`


## Error Responses

**Condition** : Authentication Fail.

**Code** : `401 BadRequest`
