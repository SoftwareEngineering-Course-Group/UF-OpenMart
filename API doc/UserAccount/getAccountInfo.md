
# Get User's Information

Get all information about an Account for the authenticated User.

**URL** : `/user/:id`

**Method** : `GET`

**Auth required** : YES

**Permissions required** : Yes

## Success Response

**Code** : `200 Success`

**Content** : `{"name":user.Name,"email":user.Email,
"phone":user.Phone}`

## Error Response

**Condition** : Authentication failed

**Code** : `401 Bad Request`
